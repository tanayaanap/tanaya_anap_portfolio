Nyaydeep - AI-Powered Constitutional Literacy & Civic Awareness Platform

Technologies: MERN Stack (MongoDB, Express.js, React.js, Node.js), PWA, Telegram API, AI Chatbot Integration

Engineered a scalable, full-stack web platform designed to democratize and simplify constitutional learning through an engaging, gamified ecosystem. The platform bridges civic knowledge gaps by blending interactive modules with intelligent, real-time user assistance.

  • Core & Gamification Features: Built and deployed interactive games, conceptual quizzes, a Telegram Bot for on-the-go access, and a dynamic dashboard for real-time progress tracking.
  • Security & Authentication: Implemented robust user authentication workflows, including secure OTP-based email verification to ensure data integrity.
  • AI & Offline Accessibility: Integrated an AI-powered chatbot for personalized learning assistance and developed Progressive Web App (PWA) capabilities to guarantee seamless offline accessibility and fluid performance.

Facial Emotion Detection using Machine Learning & Deep Learning

Technologies: Python, TensorFlow/Keras, OpenCV, Scikit-learn, FER2013 Dataset

Developed an end-to-end Computer Vision system designed to automate the recognition of seven core human emotions: Happiness, Sadness, Anger, Surprise, Fear, Disgust, and Neutrality. By leveraging the FER2013 dataset, the project bridges the gap between human sentiment and machine interpretation, providing scalable solutions for Healthcare, Online Learning, and Security systems.

  • Model Benchmarking: Conducted a rigorous performance comparison between Deep Learning (CNN) and traditional Machine Learning models (SVM, Random Forest) to optimize classification accuracy.
  • Real-Time Detection: Integrated OpenCV to enable live emotion recognition via webcam input, allowing the system to interpret facial expressions in real-time.
  • End-to-End Pipeline: Managed the full lifecycle of the project, including image preprocessing, feature extraction, model training/testing, and the development of a functional user interface for seamless interaction.

Cinema Ticket Booking System

Technologies: C, Data Structures & Algorithms (DSA)

Developed a high-performance, console-based cinema ticket booking system in C using advanced Data Structures and Algorithms. The application efficiently manages real-time theater operations by utilizing Circular Linked Lists to track sequential seating arrangements, Queues to handle user booking requests in order, and Arrays for instant matrix-based seat visualization.

  • Implemented core functionalities including automated seat reservation, dynamic ticket cancellation, and real-time seating availability displays.
  • Optimized memory management and data retrieval times by picking the right data structures for specific transactional tasks.
